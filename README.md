# GlowUp – Android App Overview

GlowUp is an Android application designed to help users build and follow a personalized skincare routine. The app allows users to view skincare steps, organize their routine, and track their skin progress over time.

The goal of this project was to demonstrate multiple screens, user interaction, and dynamic data handling using Android development tools.

## Features

- View a personalized skincare routine
- See morning and night skincare steps
- Track skin progress over time
- Add progress notes
- Navigate between screens
- Display dynamic data using RecyclerView

## Screens

### Personalized Routine Screen
Displays the user’s skincare routine, including:
- Morning routine
- Night routine
- Step-by-step product order
- Start Routine button

### Progress Tracking Screen
Shows the user’s skincare progress, including:
- Date list
- Skin condition notes
- Add Progress button

### AI Assistant Screen
This screen shows how the user can interact with an AI assistant to create a personalized skincare routine. The user shares their skin type and needs, and the app suggests suitable products and steps for morning and night. This makes the app more helpful and easier to use.

## Technologies Used

- Java
- Android Studio
- RecyclerView
- Intents for navigation between screens
- Local dynamic data
